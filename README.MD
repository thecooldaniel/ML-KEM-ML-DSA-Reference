## Setup
1. Create a new virtual environment
```Bash
cd /path/to/this/repo
python -m venv .venv
```
2. Activate the environment
```Bash
# Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate.bat
```

1. Install the required packages
```Bash
pip install -r requirements.txt
```
